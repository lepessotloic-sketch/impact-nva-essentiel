Foire aux questions

Les réponses importantes avant de commander votre site V1 Essentiel.

Quel est le délai de création ?

Votre site est prêt sous 48h après réception de vos éléments : logo, photos, coordonnées et informations de l’entreprise.

Combien coûte le domaine ?

Le nom de domaine est acheté chez OVH. Comptez environ 10 à 15 €/an. Il est enregistré à votre nom.

Comment fonctionne le paiement ?

Le paiement se fait en ligne via Stripe, solution sécurisée pour les paiements par carte.

Où est hébergé mon site ?

Votre site est publié sur GitHub Pages. L’hébergement est offert par Impact-NVA.

Puis-je modifier mon site plus tard ?

Oui. Nous pouvons ajouter de nouvelles réalisations ou mettre à jour vos informations selon vos besoins.

Le site fonctionne sur téléphone ?

Oui. Le site est responsive et pensé pour être lisible sur ordinateur, tablette et smartphone.
